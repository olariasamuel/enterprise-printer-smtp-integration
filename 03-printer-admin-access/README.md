# 03 - Printer Administrator Access

## Objective

Obtain administrative access required to modify SMTP and scan-to-email settings.

## Problem

The HP Embedded Web Server allowed access to informational pages but restricted configuration pages.

## Investigation

- Verified EWS accessibility.
- Identified administrator authentication requirement.
- Contacted support teams.
- Confirmed credentials were managed by corporate IT.

## Resolution

Obtained approved administrative access and reconfigured the device after a factory reset.

## Security Consideration

No credentials, serial numbers, or sensitive information are included in this documentation.
