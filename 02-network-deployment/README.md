# 02 - Network Deployment

## Objective

Connect the printer to the appropriate hotel network and validate connectivity.

## Actions Performed

- Validated network connectivity.
- Accessed the printer through its IP address.
- Tested communication with required services.
- Coordinated with the network provider.

## Key Finding

Basic network connectivity did not guarantee access to internal services.

The printer could communicate on the network but was not initially authorized to use the corporate SMTP relay.

## Lesson Learned

Enterprise environments frequently enforce service-level access controls beyond simple network connectivity.
